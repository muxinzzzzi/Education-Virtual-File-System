# 📋 列表功能使用指南

## 🎉 新功能已添加！

现在系统支持**论文列表查看功能**，不用再手动输入paper_id！

---

## ✨ 功能说明

### 1. **作者** - 查看我的论文列表

**位置**: 作者菜单 → [2] 查看论文状态

**使用方法**:
```
1. 选择 [2] 查看论文状态
2. 看到提示："输入 0 可列出您的所有论文"
3. 输入: 0
4. 系统显示您提交的所有论文列表
5. 从列表中选择要查看的论文ID
```

**显示内容**:
- 论文ID
- 论文标题
- 论文状态（Submitted, Under Review, Accepted等）

---

### 2. **审稿人** - 查看待审论文列表

**位置**: 审稿人菜单 → [3] 查看审稿状态

**使用方法**:
```
1. 选择 [3] 查看审稿状态
2. 看到提示："输入 0 可列出分配给您的所有论文"
3. 输入: 0
4. 系统显示分配给您的所有待审论文
5. 从列表中选择要查看的论文ID
```

**显示内容**:
- 论文ID
- 论文标题

---

### 3. **编辑** - 查看所有论文列表

**位置**: 编辑菜单 → [4] 查看审稿进度

**使用方法**:
```
1. 选择 [4] 查看审稿进度
2. 看到提示："输入 0 可列出系统中的所有论文"
3. 输入: 0
4. 系统显示所有论文的概览
5. 从列表中选择要查看详细进度的论文ID
```

**显示内容**:
- 论文ID
- 论文标题
- 作者
- 分配的审稿人
- 当前状态

## 🔄 与原有功能的兼容性

✅ **完全兼容**！原有的直接输入paper_id的方式依然有效

### 两种使用方式：

#### 方式1：直接输入paper_id（原有方式）
```
➜ 论文ID: P001  ← 直接输入
```

#### 方式2：先看列表再选择（新方式）
```
➜ 论文ID (0=列表): 0  ← 输入0查看列表
... (显示列表) ...
➜ 请输入要查看的论文ID: P001  ← 从列表选择
```

## 💡 使用技巧

### 1. 快速查看自己的论文
```
作为作者 → [2] → 0 → 查看所有论文概览
```

### 2. 查找待审论文
```
作为审稿人 → [3] → 0 → 查看分配给我的论文
```

### 3. 全局论文管理
```
作为编辑 → [4] → 0 → 查看所有论文状态
```

### 4. 配合智能记忆使用
- 系统仍然记住最近使用的paper_id
- 可以直接回车使用记住的ID
- 或输入0查看完整列表

## 📊 列表信息说明

### 论文状态含义

| 显示状态 | 含义 |
|---------|------|
| Submitted | 已提交，等待处理 |
| Under Review | 正在审稿中 |
| Accepted ✓ | 已接受 |
| Rejected ✗ | 已拒绝 |
| Major Revision | 需要大修 |
| Minor Revision | 需要小修 |

### 审稿人显示

- `(未分配)` - 还没有分配审稿人
- `bob` - 单个审稿人
- `bob,tim` - 多个审稿人
- `bob...` - 审稿人太多被截断
